# RBI 서비스 기획 문서


## 1. 사용자 페르소나 & 시나리오

**사용자 UI**
로그인 페이지가 뜬다 -> 아이디/비밀번호를 입력한다 -> 로그인 버튼을 누른다 -> 원격 바탕화면이 렌더링된다 -> 상단 바에 현재 로그인 사용자 이름과 세션 만료 타이머가 표시된다 -> 작업을 한다 -> 작업 중 만료 5분 전 경고 팝업이 뜬다 -> 세션 연장 버튼을 누른다 -> 타이머가 초기화된다 -> 작업을 마치고 상단 바의 내 계정 메뉴를 연다 -> 세션 목록에서 다른 기기의 활성 세션을 확인하고 종료한다 -> 비밀번호 변경 메뉴에서 새 비밀번호를 설정한다 -> 로그아웃 버튼을 눌러 세션을 종료한다

**프린터**
컴퓨터를 킨다 -> 브라우저를 연다 -> 링크 주소를 친다 -> 사용자 인증을 위해 로그인 페이지가 뜬다 -> 로그인을 한다 -> 바탕화면이 보인다 -> 접속하고 싶은 애플리케이션을 실행한다 -> 프린트를 시도한다 -> 로컬 프린터로 프린트


## 2. 기능 명세

**사용자 UI**
- 현재 로그인 사용자 표시
- 세션 만료 타이머
  - 만료 N분 전 경고 알림 표시
  - 세션 연장 버튼
- 비밀번호 변경
- 세션 목록
  - 활성 세션 상세 정보 표시 (접속 IP, 시작 시각, 유휴 시간)
  - 특정 세션 강제 종료
  - 전체 세션 일괄 종료
- 로그아웃 버튼
- 현재 세션 종료 버튼
